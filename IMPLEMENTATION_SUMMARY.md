## üîß Customization Tips

### To Change Colors Globally
Edit the CSS variables in `styles.css` lines 1-40:
```css
--first-color: hsl(184, 71%, 39%);      /* Primary color */
--accent-color: hsl(14, 100%, 53%);     /* Accent color */
```

### To Reorder Sections
Move the corresponding `<section>` blocks in `index.html` while keeping navigation menu in sync.

### To Add New Skills
Add in Skills section with the format:
```html
<div class="skills__data">
    <div class="skills__names">
        <i class='bx bxl-icon-name skills__icon'></i>
        <span class="skills__name">Skill Name</span>
    </div>
    <div class="skills__bar skills__new-skill"></div>
    <div>
        <span class="skills__percentage">XX%</span>
    </div>
</div>
```

Then add CSS in `styles.css`:
```css
.skills__new-skill { width: XX%; }
```
